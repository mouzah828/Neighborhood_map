#Neighborhood map project
##This project is a webpage showing a map pointing to the list of the famous Airport locations in United Arab Emirates,
you will find more information about this location if you click the marker

##No need for any special configration to run this project, you just need to open the index.html page in any of your browser and have internet connection in your PC
##Opening the index.html  in your browser will display a page of two part:
  1. Left Side: with the list of the famous Airport names with search box to filter this names
  2. Right Side: Google Map of the United Arab Emirates showing a marker of the famous Airport locations which listed in the left side

##You will see Information Box with more information about the selected loction whenever you click on both:
  1. Any of the marker on the map
  2. Any of location names from the search list

##I used two API to dispaly the conetent of the Information Box of each location:
	1. Foursquare's API to dispaly name,address,more information link for redirect the user to Foursquare website to read more about this location
	2. yahooapis API to dispaly current temprature of each location
